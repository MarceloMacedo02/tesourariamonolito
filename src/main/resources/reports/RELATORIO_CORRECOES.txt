# CORREÇÕES REALIZADAS NOS RELATÓRIOS JASPERREPORTS

## Resumo das correções

### 1. Problema Crítico Resolvido
- **Erro:** `net.sf.jasperreports.engine.JRRuntimeException: Could not find subdataset named "MovimentoDataset"`
- **Solução:** Adicionada a declaração do subDataset no arquivo `subreport_rubrica.jrxml`
- **Arquivo afetado:** `subreport_rubrica.jrxml`

### 2. Avisos de Depreciação Eliminados
- **Aviso:** `The 'isStretchWithOverflow' attribute is deprecated. Use the 'textAdjust' attribute instead.`
- **Solução:** Substituídos por `textAdjust="StretchHeight"`
- **Arquivos afetados:**
  - `subreport_rubrica.jrxml` (2 ocorrências)
  - `demonstrativo_financeiro_mensal_report.jrxml` (1 ocorrência)
  - `relatorio_inadimplentes.jrxml` (1 ocorrência)

### 3. Erros de Validação XML Resolvidos
- **Erro:** `cvc-complex-type.2.4.a: Foi detectado um conteúdo inválido...`
- **Solução:** Corrigida a estrutura do XML com posicionamento adequado do subDataset
